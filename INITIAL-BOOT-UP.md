# Initial boot up steps

> [!NOTE]
> What actions do I take when starting up a new Mac for the first time?

## Initial boot up

- **Language:** English
- **Select your country or region:** United States
- **Transfer your data to this Mac:** Set up as new
- **Accessibility:** Not Now
- Setup my wi-fi network
- **Create a Mac account:**
    - Full name
    - Nickname for account name
    - Allow password to be changed by Apple account
- **Sign in to your Apple account**
    - Agree to T&C
- **Enable location services**: enable
- **Select your time zone:** set automatically
- **Analytics:** do not share
- **Screen time:** setup later
- **Apple intelligence:** setup later
- **Siri:** disable Ask Siri
- **Touch ID:** setup
- **Apple Pay:** setup later
- **Choose your look:** Dark
- **Update Mac automatically:** default

## First login

- Allow calendar to use location.
- **Remove from dock:**
    - Messages
    - Mail
    - Maps
    - Facetime
    - Phone
    - Contacts
    - Reminders
    - Notes
    - Apple TV
    - Keynote
    - Numbers
    - Pages
    - Games
- **Configure `Finder`:**
    - `General > Show these items on desktop`: Disable external disks and CDs, etc.
    - **Disable in `Sidebar`:**
        - Recents
        - Shared
        - iCloud Drive
        - iCloud Storage
        - Bonjour computers
        - Connected servers
        - Recent tags

## Apply any OS updates

> [!NOTE]
> While any updates are downloading / preparing to install, I typically run through as much of
> the [system settings changes](./SYSTEM-SETTINGS.md) I can.

- `System Settings > General > Software Update > Update Now`
- Repeat until all updates applied.
